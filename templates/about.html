About the Project

Project Overview

This web-based attendance system was developed as a final year capstone project to digitize and streamline attendance tracking in educational institutions.

The system provides role-based access for administrators, teachers, and students, allowing for efficient management and monitoring of attendance records. It simplifies the process of marking attendance, generating reports, and tracking student presence across different subjects and classes.

Our solution offers a user-friendly interface with modern design principles, ensuring that users of all technical backgrounds can navigate and use the system with ease.

Technology Stack

  • HTML/CSS/JavaScript Frontend
  • Python (Flask) Backend
  • SQLite Database
  • Bootstrap UI Framework
